PropertyDataType

The type of data stored inside the property. The exact type this refers to depends on the PropertyType:

  • ValueThe type of value that is defined. E.G. if the property public string X { get; set; } was defined on the node, the PropertyDataType would be string.
  • ExpressionThe type of value that is expected by the expression. E.G. if the property public ExpressionReference<double> was defined, PropertyDataType would be double.
  • StatementAlways null.
  • VariableThe type of value that is allowed by the variable reference. E.G. if the property public VariableReference<bool> was defined, PropertyDataType would be bool.
